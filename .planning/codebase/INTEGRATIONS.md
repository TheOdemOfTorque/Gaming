## APIs & External Services

- Phaser 3 — JavaScript game framework for `prime-empire/`

## Data Storage

**Databases:**
- None — no backend database

**Browser Storage (localStorage):**
- `1x1-trainer/`: key `henry_einmaleins` — stores all game state (name, XP, achievements, Leitner boxes, highscores, stats); `STATE_VERSION = 4` with migration chain for schema updates
- `1x1-trainer/`: key `sb_settings` — stores Space Blaster settings (a sub-game embedded in `1x1-trainer/index.html`)
- `space-invader-revibed/`: key `sb_settings` — stores settings (same pattern as above)
- High score lists stored per-difficulty via `localStorage.setItem(hsKey(d), ...)` in both `1x1-trainer/index.html` and `space-invader-revibed/index.html`

**File Storage:**
- Local filesystem only (`1x1-trainer/icon-192.png`, `1x1-trainer/icon-512.png` generated by `scripts/generate-icons.js`)

**Caching:**
- Service Worker Cache API (`1x1-trainer/sw.js`) — cache name `1x1-trainer-v18`; caches all PWA assets for offline play
- No other caching layer

## Authentication & Identity

**Auth Provider:**

## Browser APIs Used

**Web Audio API:**
- `1x1-trainer/index.html` lines 1072+ — `AudioContext`/`webkitAudioContext`; synthesizes all sounds (correct answers, combos) using oscillators and gain nodes; no audio files
- `space-invader-revibed/index.html` lines 270, 276, 313 — same pattern: `AudioContext`/`webkitAudioContext`, `createOscillator()`, `createGain()`; synthesized sound effects only

**Canvas 2D API:**
- `1x1-trainer/index.html` — `#bg-canvas` for twinkling star background animation (line 1093), `#c` canvas for embedded Space Invader sub-game
- `space-invader-revibed/index.html` — `<canvas id="c">` for full game rendering
- `vibe-coding-demo-4-timm/index.html` — `<canvas id="game">` for full game rendering
- `prime-empire/index.html` — Phaser 3 manages canvas internally via `<div id="game">`

**Service Worker API:**
- `1x1-trainer/` only — `navigator.serviceWorker.register('./sw.js')` in `1x1-trainer/index.html` line 3303-3305; cache-first strategy for offline PWA support

**requestAnimationFrame:**
- `1x1-trainer/index.html` — star background loop (line 1112), Space Invader game loop (lines 3200-3202)
- `space-invader-revibed/index.html` — main game loop
- `vibe-coding-demo-4-timm/index.html` — main game loop
- `prime-empire/index.html` — managed by Phaser 3
